Gives you the posibility to share links with Google Analytics tracking Code.

== Description ==

I like tracking of urls all over the web. Whether I'm sharing on Facebook or Google+ or where ever.
But it was to many work for me to create this links with all the parameters. So I decidet to create this plugin.

You can add as many sources (Pages where to share your links) as you like by a admin page. The Links will show to the writer at the live side of your page.

After sharing the links in the internet you can track the clicks on Google Analytics

== Installation ==

1. Upload `tracking-url` folder to the `/wp-content/plugins/` directory or install it by the wordpress repo
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the settings page and add new sources
